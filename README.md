Названия проекта: TalentFlow - Поток талантов

Описание предметной области по вашим таблицам: это комплексная система управления наймом персонала (Applicant Tracking System - ATS), предназначенная для автоматизации полного цикла рекрутинга в современных компаниях. Система охватывает все этапы — от создания вакансии до принятия кандидата на работу, обеспечивая прозрачность, эффективность и аналитику процессов подбора персонала.
